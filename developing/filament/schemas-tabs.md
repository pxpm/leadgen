# Schema Tabs (Filament v5)
**URL:** https://filamentphp.com/docs/5.x/schemas/tabs
`Tabs::make('Tabs')->tabs([Tab::make('Tab1')->schema([...]),Tab::make('Tab2')->schema([...])])`
`Tab::make('Profile')->icon('heroicon-o-user')` `Tab::make('Comments')->badge(5)->badgeColor('danger')`
`->activeTab(1)` `->persistTabInQueryString('tab')` `->tabPosition(Position::Start)`
